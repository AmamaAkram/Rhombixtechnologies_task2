# Task 1 — Daraz Homepage Redesign
This project is a modern redesign of the homepage of Daraz.
The goal of this redesign was to improve the overall user experience by creating a cleaner, more organized, and user-focused interface. The original homepage contained excessive visual clutter, inconsistent spacing, and overwhelming promotional content. This redesign focuses on improving usability, readability, spacing, visual hierarchy, and product discoverability while maintaining the brand identity of Daraz.

# The redesigned homepage includes:
Modern navigation bar,
Improved search experience,
Cleaner hero banner,
Organized category section,
Structured product grids,
Better spacing and alignment,
Responsive and minimal layout,
Improved footer design,
# Problems Identified
Crowded homepage layout,
Too many promotional banners,
Weak visual hierarchy,
Inconsistent spacing,
Overuse of bright colors,
Complex navigation structure,
Poor content organization,
# Improvements Made
Simplified UI layout,
Added balanced white space,
Improved typography and readability,
Organized product cards into clean grids,
Enhanced visual hierarchy,
Better alignment and spacing,
Cleaner category buttons,
More user-friendly shopping experience,
# Tools Used
Figma,
UI/UX Design Principles,
Wireframing & Layout Planning



# Task 2 — Food Delivery App Wireframes
This project includes low-fidelity wireframes for a food delivery mobile application. The purpose of the wireframes was to plan the user flow, layout structure, and screen organization before moving to high-fidelity UI design.
The wireframes focus on creating a simple, intuitive, and user-friendly experience for customers ordering food online.

# Screens Included
Splash Screen,
Login / Signup,
Home Screen,
Restaurant Listing,
Food Details,
Cart Screen,
Checkout Screen

# Objectives
Improve user navigation,
Create clear screen hierarchy,
Simplify ordering process,
Ensure smooth user flow,
Design mobile-friendly layouts,
Features Planned,
Search functionality,
Food categories,
Restaurant cards,
Add to cart,
Order tracking,
Secure checkout

# Tools Used
Figma,
Wireframing Techniques,
UX Planning Principles
